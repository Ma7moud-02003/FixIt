# 🔧 Fixit - Graduation Project

Fixit is a graduation project that aims to connect users with skilled technicians for home services such as plumbing, electrical work, and maintenance in a fast and reliable way.

---

## 🎓 Project Overview

Fixit helps users easily find and hire trusted technicians, while providing a smooth experience for requesting services and managing interactions.

---

## 🚀 Live Demo

* 🌐 Frontend:https://fix-it-bay.vercel.app/home
* 📡 API (Swagger): https://fixitapi.runasp.net/swagger/index.html

---

## 🧩 Features

* 👤 User authentication (Register / Login)
* 🔍 Browse technicians
* 🛠️ Request home services
* ⭐ Rating and feedback system
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** Angular
* **Styling:** Tailwind CSS
* **Backend:** .NET REST API
